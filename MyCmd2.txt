Branching ------
git branch "branch name"    - create branch

git checkout <new branch name>  - Checkout Branch

git merge "<branch Name>"  - merge

git branch -d "<branch Name>"   - delete branch locally
git push origin --delete "<branch Name>" - delete branch remote

Add tag----
git tag v1.0 - create tag
git tag      -- to list tags

git push origin v1.0   -- push tag to repository


git tag -d v1.0  -- delete tag locally
git tag --delete v1.0  -- delete tag locally

git oush origin -d v1.0  -- delete tag repository
